# RShell
在Unity游戏开发的过程中，我们常常会有这样的调试需求，能在运行时（包括真机）查看一些属性值，调用一些函数，RShell是解决这个问题的一个小工具。
https://zhuanlan.zhihu.com/p/649662811

## 需要python3环境，使用pip安装依赖库
```Shell
pip install prompt_toolkit
pip install pyfiglet
```
